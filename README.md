# WorkData

> 个人代码工作数据存储与备份仓库

## 简介

本仓库用于存储个人编程学习过程中的代码片段、配置文件、开发笔记及相关资源文件。作为开发工作流的辅助工具，所有内容均为个人技术积累与项目备份。

## 使用说明

1. **Configs**: 包含编辑器配置、环境变量模板等（已脱敏处理）
2. **Snippets**: 按语言/框架分类的常用代码片段
3. **Notes**: Markdown格式的技术学习笔记，部分包含配图说明
4. **Assets**: 笔记中引用的本地资源文件，**仅用于个人文档渲染**
5. **Archives**: 已停止维护的旧项目备份

## 技术栈

- 主要语言: Python, JavaScript, Shell
- 文档格式: Markdown
- 版本管理: Git
